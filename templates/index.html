Getting Started

The MCP Assessor Agent API provides a secure interface for querying real estate assessment data. It supports multiple database types, parameterized queries, and natural language query translation.

API base URL: http://localhost:8000

API Key Authentication

All API requests require an API key for authentication:

curl -X GET http://localhost:8000/api/health \ -H "X-API-Key: your_api_key_here"

For testing, use the default API key from your .env file.

Health Check

Verify the API is running and database connections are healthy:

GET /api/health
